# Onshape-Dot
AutoHotkey Utility Script

Changes the Numpad Comma into a Dot for Onshape.

1. Download Onshape-Dot.exe 

Double-click the executable file.

The script will begin running immediately in the background and you will see its icon appear in your system tray (near the clock).

2. Run at System Startup (Recommended)

To ensure this script automatically runs every time you log into Windows, you need to place the executable in your Startup folder:

Press Win + R to open the Run dialog.

Type shell:startup and press Enter. This will open the designated Startup folder for the current user.

Copy and paste the Onshape-Dot.exe file into this folder.

The next time you start your computer, the script will launch automatically.
